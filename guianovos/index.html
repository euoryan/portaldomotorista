Guia Completo para Novos Motoristas

Manual completo passo a passo para acesso ao sistema de comunicação e recebimento de rotas. Siga este guia para garantir uma integração eficiente.

Configuração Inicial

Bem-vindo à nossa equipe de motoristas! Para uma comunicação eficiente e recebimento de rotas programadas, é necessário configurar o Telegram seguindo este guia passo a passo.

1
Instale o Telegram

Primeiro, você precisa baixar e instalar o aplicativo Telegram em seu dispositivo móvel. Selecione a opção de acordo com seu sistema operacional:

Android iOS (iPhone)
Dica: O Telegram é um aplicativo leve e gratuito. Certifique-se de ter uma conexão estável à internet durante a instalação. Caso já tenha o aplicativo instalado, verifique se está atualizado na versão mais recente.
2
Inicie o Bot de Rotas

O próximo passo é iniciar o Bot de Rotas, que será responsável por enviar automaticamente suas rotas programadas direto para seu Telegram:

  1. Clique no botão abaixo para acessar o Bot de Rotas
  2. Quando o Telegram abrir, pressione o botão "Iniciar" ou "Start"
  3. O bot responderá com uma mensagem de boas-vindas e seu ID de usuário
Iniciar Bot de Rotas
Importante: Este bot serve apenas para o envio automatizado de suas rotas planejadas. Não é um canal direto de comunicação com nossa equipe. Todas as mensagens do bot terão um botão de suporte que o direcionará para o atendimento operacional.
3
Contate o Suporte Operacional

Para finalizar seu cadastro, entre em contato com o Suporte Operacional e informe seu nome de guerra:

  1. Clique no botão abaixo para iniciar a conversa com o suporte
  2. Envie uma mensagem com seu nome de guerra (aquele definido durante sua contratação)
  3. Aguarde a confirmação do setor operacional
Contatar Suporte Operacional
Importante: O suporte operacional é seu canal principal de comunicação. Utilize-o para todas as dúvidas relacionadas às rotas, informações detalhadas, resolução de problemas ou outras necessidades. Mantenha sempre uma comunicação profissional e respeitosa.

Comunicação Diária

Após a configuração inicial, siga estas diretrizes para uma comunicação eficiente:

  • Verifique o Telegram regularmente para receber suas rotas programadas
  • Confirme o recebimento das rotas quando solicitado
  • Reporte imediatamente ao suporte operacional qualquer problema ou dúvida
  • Mantenha seus dados de contato sempre atualizados

Para esclarecimentos adicionais ou suporte técnico, entre em contato com o departamento operacional através do canal oficial: Suporte Operacional
