# Portal do Motorista

Este é o **Portal do Motorista**, um projeto desenvolvido para facilitar o acesso a recursos, guias e orientações sobre regulamentações, que podem ser utilizados por motoristas para otimizar sua experiência na estrada.

## Sobre
O "Portal do Motorista" é uma plataforma com o objetivo de fornecer um diretório de facilidades e informações úteis para motoristas, incluindo regulamentações e outros recursos essenciais para a atividade. O projeto foi criado de forma intuitiva e responsiva, permitindo que qualquer motorista ou empresa utilize e compartilhe essas informações facilmente.

## Destaques
- Design direto e funcional
- Totalmente responsivo
- Navegação intuitiva
- Fácil acesso a informações e guias

## Tecnologias
- **HTML**: Estrutura do site
- **CSS**: Estilo e layout responsivo
- **JavaScript**: Funcionalidades interativas

## Como usar
1. Clone o repositório:
   ```bash
   git clone https://github.com/euoryan/portaldomotorista
   ```
2. Acesse o diretório do projeto:
   ```bash
   cd portaldomotorista
   ```
3. Abra o arquivo `index.html` no navegador para visualizar o portal.
